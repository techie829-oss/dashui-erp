# MyDash "Pro Max" Design System

## 1. Visual Theme & Atmosphere
- **Style**: Data-Dense Dashboard / Professional Minimalist
- **Mood**: Precise, Technical, Analytical, Efficient
- **Philosophy**: "Data First." High-density information with maximum readability and stroke consistency.

## 2. Color Palette (Semantic Tokens)
| Role | Hex | Variable | Usage |
|------|-----|----------|-------|
| **Primary** | `#0F172A` | `$primary` | Main brand, active states, deep headers |
| **Secondary** | `#1E293B` | `$secondary` | Secondary buttons, nav background |
| **Accent / CTA** | `#22C55E` | `$success` | Primary actions, positive trends, growth |
| **Background** | `#F8FAFC` | `$bg-secondary` | Page background (Light Mode) |
| **Surface** | `#FFFFFF` | `$bg-primary` | Cards, Sidebar, Navbar (Light Mode) |
| **Text Primary**| `#0F172A` | `$text-primary` | Headings, main numbers |
| **Text Muted** | `#64748B` | `$text-muted` | Labels, axis text, secondary info |
| **Border** | `#E2E8F0` | `$border-default`| Dividers, input borders |

## 3. Typography
- **Heading/Brand**: `Fira Sans`, Semibold (600)
- **Body Text**: `Fira Sans`, Regular (400)
- **Data/Numbers**: `Fira Code`, Medium (500)
- **Scale**: 12px (XS), 14px (SM), 16px (Base), 20px (LG), 24px (XL)

## 4. Components & Motion
- **Icons**: Lucide Icons (2px Stroke Width).
- **Cards**: Solid White, 1px Border (`#E2E8F0`), Subtle Shadow (`$shadow-sm`).
- **KPI Cards**: Includes Trend Indicators (Percentage + Directional Arrow).
- **Buttons**: Rounded (8px), Solid or Outline with 200ms ease transitions.

## 5. Layout Principles
- **Grid**: 8px Base Spacing System.
- **Max Width**: 1440px (Centered).
- **Density**: Compact (1.2x information density vs standard layouts).
